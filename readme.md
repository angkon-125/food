# Food Project Website

A responsive food ordering website with a reservation system and dynamic cart functionality.

## Features

- Responsive navigation menu
- Dynamic menu with order buttons
- Cart functionality connected to backend
- Reservation form submission
- Image gallery and section animations
- Mobile-friendly and fully responsive design

## Setup Instructions

### Frontend
1. Open `frontend/index.html` in your browser.
2. Ensure the `css/` and `img/` folders are correctly linked.

### Backend
1. Install Node.js if not already installed.
2. Navigate to the `backend/` folder in your terminal.
3. Install dependencies and start the server:

```bash
npm init -y
npm install express cors
node server.js
